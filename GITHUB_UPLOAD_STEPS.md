# GitHub Hochladen

## Variante Ohne Git-Programm

1. Bei GitHub anmelden.
2. Oben rechts auf `+` klicken.
3. `New repository` auswaehlen.
4. Repository-Name zum Beispiel:

   ```text
   balkonkraftwerk-kombi-display
   ```

5. Sichtbarkeit waehlen:
   - `Public`, wenn es oeffentlich sein soll
   - `Private`, wenn du erst testen willst

6. Keine README, keine License und keine .gitignore von GitHub erzeugen lassen, weil diese Dateien hier schon vorbereitet sind.
7. Repository erstellen.
8. Auf `uploading an existing file` klicken.
9. Den Inhalt dieses `GitHub`-Ordners hochladen.
10. Commit-Text zum Beispiel:

   ```text
   Erste Hotspot-Version
   ```

## Release Erstellen

1. Im Repository rechts auf `Releases` klicken.
2. `Create a new release` auswaehlen.
3. Tag:

   ```text
   v1.0.0
   ```

4. Titel:

   ```text
   Balkonkraftwerk Kombi-Display v1.0.0
   ```

5. Text aus `RELEASE_TEXT.md` einfuegen.
6. Als Datei anhaengen:

   ```text
   firmware/balkonkraftwerk-kombi-display-hotspot-v1-factory.bin
   ```

7. Release veroeffentlichen.
