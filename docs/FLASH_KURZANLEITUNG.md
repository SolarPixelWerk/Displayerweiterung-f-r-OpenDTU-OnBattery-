# Flash-Kurzanleitung

1. Chrome oder Edge oeffnen.
2. Diese Seite aufrufen:

   ```text
   https://web.esphome.io/?dashboard_install
   ```

3. ESP32-2432S028R per USB verbinden.
4. Auf `CONNECT` klicken.
5. Seriellen Port auswaehlen.
6. Datei auswaehlen:

   ```text
   firmware/balkonkraftwerk-kombi-display-hotspot-v1-factory.bin
   ```

7. Installation starten.
8. Nach dem Neustart mit dem WLAN `Balkonkraftwerk-Setup` verbinden.
9. Im Browser `http://192.168.4.1` oeffnen.
10. WLAN- und OpenDTU-OnBattery-Daten eintragen.
